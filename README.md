# ✦ Emilly "Emy" Atsuya — Perfil de Personagem

> *"Compreensão é só uma questão de perspectiva."*

Página web atmosférica de perfil de personagem para **Emilly "Emy" Atsuya**, feiticeira maldita de um sistema de RPG de mesa inspirado em Jujutsu Kaisen.

---

## ✨ Visão Geral

Uma página de perfil dark-themed, altamente atmosférica, com estética occult/Jujutsu Kaisen. Construída com React + TypeScript e animada com Framer Motion.

### Seções da Página

- **Hero** — Nome em Cinzel gothic, placeholder de retrato com anéis de sigilo rotativos, e card glassmorphism com a frase icônica
- **Atributos** — Grid responsivo de 6 cards com ícones Lucide React e efeitos hover
- **Lore — Capítulo 1: "Existência"** — Texto narrativo com dois estilos visuais distintos: voz da Emy (itálico teal) vs narração (serif muted)

## 🚀 Como Rodar

```bash
# Instalar dependências
npm install

# Servidor de desenvolvimento
npm run dev

# Build de produção
npm run build
```

O servidor abre em **http://localhost:5173/**
